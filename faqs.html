RM MANIFEST TOOL OPEN BETA
FAQ / Dúvidas Frequentes
O RM Manifest Tool é um app para Windows/Linux/Mac que facilita o download e gerenciamento de arquivos .manifest e .lua/.st de jogos Steam. Ele automatiza tarefas como:
  • Buscar AppIDs e baixar rapidamente arquivos originais do Steam.
  • Salvar esses arquivos em pastas customizadas do seu PC.
  • Atualizar arquivos já baixados com um clique.
  • Facilitar a manipulação para projetos de tradução, mods, backups ou estudos.
Você não precisa usar comandos do SteamCMD nem procurar arquivos manualmente — o app faz tudo por você, com interface amigável!
Siga estes passos simples:
  1. Digite o App ID ou nome do jogo no campo de busca (ex: 271590 ou "GTA V").
  2. Selecione o jogo correto na lista de sugestões.
  3. Se o jogo tiver branches, escolha qual deseja.
  4. Clique em ADD — os arquivos serão baixados e copiados para as pastas configuradas.
Dica: Você pode buscar jogos pelo nome completo ou parcial (ex: “Far Cry”, “Counter” etc.).
Use os botões de pasta ao lado de cada campo:
  • LOCALIZAÇÃO .ST/.LUA: onde os arquivos .st ou .lua serão copiados.
  • LOCALIZAÇÃO DO .MANIFEST: onde os arquivos .manifest vão ser salvos.
Você pode escolher qualquer pasta do seu computador. Se não escolher, o app usa o caminho padrão do Steam.
Dica: Para restaurar para o padrão, basta apagar o campo e clicar novamente no botão de pasta.
Quando um AppID aparece com o aviso “possui proteção DRM”, significa que os arquivos desse jogo possuem criptografia, autenticação ou outros bloqueios feitos pela Steam.
Nesses casos:
  • Os arquivos .manifest podem não funcionar fora do Steam.
  • Você pode receber erro ao tentar abrir ou usar os arquivos baixados.
  • Nem todos os jogos têm DRM — muitos arquivos funcionam normalmente.
O app apenas alerta, mas permite o download caso você queira tentar usar por conta própria.
Isso pode acontecer por:
  • AppID digitado errado (confira se é o ID certo do jogo na Steam).
  • O branch escolhido não existe ou está vazio (ex: “beta”, “experimental” etc).
  • O jogo ainda não foi adicionado ao servidor ou não tem manifest público.
Solução: Verifique o AppID, tente trocar o branch para “public” e confira se o jogo está disponível na Steam.
Se persistir, mande um pedido de suporte.
Clique em ATUALIZAR TUDO para o app buscar os manifests mais recentes para o AppID selecionado e baixar novamente, substituindo os antigos.
Importante: Só funciona para branch “public” ou jogos suportados. Se não houver arquivos novos, será avisado!
Sempre que você abrir o app, ele verifica se existe uma versão nova:
  • Se houver, baixa e instala automaticamente — você só precisa aguardar o carregamento inicial (splash).
  • Se não houver, o app abre normalmente.
Não é necessário baixar manualmente do site, só aguardar a atualização automática.
O botão REINICIE STEAM tenta fechar e reabrir o Steam para você automaticamente, detectando o sistema operacional (Windows, Linux, Mac).
Se não funcionar:
  • No Windows: será solicitado para você escolher o Steam.exe manualmente.
  • Em outros sistemas: feche e abra o Steam manualmente.
Se o Steam não iniciar, reinicie seu computador.
